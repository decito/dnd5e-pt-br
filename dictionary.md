# DICIONÁRIO

## Tabela 1: Glossário de Termos Essenciais (Inglês vs. Português Brasileiro)

Esta tabela serve como um recurso central para a revisão das strings, consolidando os termos-chave, suas traduções recomendadas e as justificativas por trás dessas escolhas, com referências às fontes de pesquisa.

| Termo em Inglês (Original) | Termo em Português Brasileiro (Recomendado) | Abreviação | Fonte Principal        | Justificativa/Notas                                                                                              |
| -------------------------- | ------------------------------------------- | ---------- | ---------------------- | ---------------------------------------------------------------------------------------------------------------- |
| Ability Check              | Teste de Atributo                           | -          | Oficial WotC/Galápagos | Mantém consistência com "Atributo".                                                                              |
| Ability                    | Atributo                                    | -          | Oficial WotC/Galápagos | Distingue de "Habilidade" (Skill), eliminando ambiguidade.                                                       |
| Advantage                  | Vantagem                                    | -          | Oficial WotC/Galápagos | Termo padrão e amplamente aceito para rolagem de dados favorável.                                                |
| Adventure                  | Aventura                                    | -          | Oficial WotC/Galápagos | Termo padrão para uma série de encontros que contam uma história.                                                |
| Armor Class                | Classe de Armadura                          | CA         | Oficial WotC/Galápagos | Termo padrão e amplamente reconhecido.                                                                           |
| Attack Roll                | Jogada de Ataque                            | -          | Oficial WotC/Galápagos | Diferencia de "Teste" e evita conotações indesejadas do verbo "rolar".                                           |
| Background                 | Antecedente                                 | -          | Oficial WotC/Galápagos | Termo padrão para a história pregressa do personagem.                                                            |
| Cantrip                    | Truque                                      | -          | Oficial WotC/Galápagos | Termo consolidado para magia de nível zero conjurada à vontade.                                                  |
| Challenge Rating           | Nível de Desafio                            | ND         | Oficial WotC/Galápagos | Termo padrão para medir a dificuldade de criaturas e encontros.                                                  |
| Class                      | Classe                                      | -          | Oficial WotC/Galápagos | Termo padrão para a definição de classes de personagens.                                                         |
| Condition                  | Condição                                    | -          | Oficial WotC/Galápagos | Termo padrão para estados que afetam criaturas.                                                                  |
| Creature                   | Criatura                                    | -          | Oficial WotC/Galápagos | Termo geral para qualquer ser vivo no jogo.                                                                      |
| Damage Roll                | Jogada de Dano                              | -          | Oficial WotC/Galápagos | Diferencia de "Teste" e evita conotações indesejadas do verbo "rolar".                                           |
| Disadvantage               | Desvantagem                                 | -          | Oficial WotC/Galápagos | Termo padrão e amplamente aceito para rolagem de dados desfavorável.                                             |
| Dungeon                    | Masmorra                                    | -          | Oficial WotC/Galápagos | Termo padrão para local de aventura fechado.                                                                     |
| Dungeon Master             | DM / Mestre de Jogo                         | DM / MdJ   | Oficial WotC/Galápagos | "DM" preferencial no texto para reforçar a marca; "Mestre de Jogo" ou "Mestre" são sinônimos aceitos.            |
| Dungeon Master's Guide     | Guia do Mestre                              | -          | Oficial WotC/Galápagos | Título oficial do livro de regras para o DM.                                                                     |
| Encounter                  | Encontro                                    | -          | Oficial WotC/Galápagos | Termo padrão para uma cena em uma aventura.                                                                      |
| Feat                       | Talento                                     | -          | Oficial WotC/Galápagos | Termo padrão para habilidades especiais adquiridas.                                                              |
| Feature                    | Característica                              | -          | Oficial WotC/Galápagos | Evita confusão com "Talento". Usado para habilidades intrínsecas de Classe ou Raça.                              |
| Fiend                      | Ínfero                                      | -          | Comunidade Consolidada | Diabo seria o termo oficial, mas é inconsistentemente usado para várias descrições de magias e características.  |
| Hit Points                 | Pontos de Vida                              | PV         | Oficial WotC/Galápagos | Termo clássico, consolidado desde edições anteriores.                                                            |
| Homebrew                   | Homebrew / Conteúdo Caseiro                 | -          | Comunidade Consolidada | Em geral, mantém-se "Homebrew" no jargão. "Conteúdo caseiro" é tradução explicativa usada em guias e glossários. |
| Houserule                  | Regra de Casa                               | -          | Comunidade Consolidada | Termo padrão para regras modificadas pelo DM.                                                                    |
| Initiative                 | Iniciativa                                  | -          | Oficial WotC/Galápagos | Termo padrão para ordem de combate.                                                                              |
| Ki                         | Chi                                         | -          | Oficial WotC/Galápagos | Termo padrão para a energia espiritual dos monges.                                                               |
| Level                      | Nível                                       | -          | Oficial WotC/Galápagos | Usado tanto para Personagens quanto para Magias.                                                                 |
| Long Rest                  | Descanso Longo                              | -          | Oficial WotC/Galápagos | Tradução oficial para período de descanso prolongado.                                                            |
| Magic Item                 | Item Mágico                                 | -          | Oficial WotC/Galápagos | Tradução direta e clara.                                                                                         |
| Melee Range                | Alcance Corpo a Corpo                       | -          | Comunidade Consolidada | Termo padrão para a distância de ataques próximos.                                                               |
| Module                     | Módulo                                      | -          | Comunidade Consolidada | Usado para aventuras publicadas ou expansões menores.                                                            |
| Monster Manual             | Manual dos Monstros                         | -          | Oficial WotC/Galápagos | Título oficial do livro. "Livro dos Monstros" aparece em algumas edições, mas não é o termo preferencial.        |
| Nonplayer Character (NPC)  | Personagem Não Jogador                      | PNJ        | Oficial WotC/Galápagos | Tradução oficial e abreviação consolidada.                                                                       |
| Object                     | Objeto                                      | -          | Oficial WotC/Galápagos | Termo padrão para coisa não-viva e distinta.                                                                     |
| Player Character           | Personagem Jogador                          | PJ         | Comunidade Consolidada | Tradução padrão na comunidade, equivalente a "PC" no inglês.                                                     |
| Player's Handbook          | Livro do Jogador                            | -          | Oficial WotC/Galápagos | Título oficial do livro principal de regras para jogadores.                                                      |
| Proficiency                | Proficiência                                | -          | Oficial WotC/Galápagos | Termo padrão. Uso de preposições: "em" (habilidades), "com" (ferramentas).                                       |
| Race                       | Raça                                        | -          | Oficial WotC/Galápagos | Termo tradicional para linhagem do personagem. Nota: tendência oficial futura para "Espécie".                    |
| Range (general)            | Alcance                                     | -          | Oficial WotC/Galápagos | Usado para magias, armas e habilidades de ataque à distância.                                                    |
| Round                      | Rodada                                      | -          | Comunidade Consolidada | Complemento a "Vez". Define a passagem do tempo em combate.                                                      |
| Saving Throw               | Salvaguarda                                 | -          | Oficial WotC/Galápagos | Termo técnico para evitar confusão com "Teste" ou "Jogada". Inicialmente polêmico, mas consolidado.              |
| Short Rest                 | Descanso Curto                              | -          | Oficial WotC/Galápagos | Tradução oficial para período de descanso breve.                                                                 |
| Skill Check                | Teste de Perícia                            | -          | Oficial WotC/Galápagos | Usa "Perícia" em vez de "Habilidade" para diferenciar de Ability Score.                                          |
| Spell                      | Magia                                       | -          | Oficial WotC/Galápagos | Termo padrão para ações mágicas.                                                                                 |
| Spell Save DC              | CD de Salvaguarda                           | -          | Oficial WotC/Galápagos | "CD" para Classe de Dificuldade; uso oficial.                                                                    |
| Spell Slot                 | Espaço de Magia                             | -          | Oficial WotC/Galápagos | Tradução direta e clara para o recurso de conjuração.                                                            |
| Turn                       | Vez                                         | -          | Comunidade Consolidada | Tradução direta e amplamente aceita no contexto de jogos de turno.                                               |
